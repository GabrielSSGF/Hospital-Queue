## Key Features

* Priority Queue
  -  Includes the use of three queues, each of them with its respective priority level. You can determine up to 15 patients in line to be attended and determine their health condition. This will automatically determine the order they will be attended.

## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com) and [GCC](https://gcc.gnu.org/). From your command line:

```bash
# Clone this repository
$ git clone https://github.com/GabrielSSGF/Hospital-Queue

# Go into the repository
$ cd Hospital-Queue

# Compile the files
$ gcc -o priority_list main.c src/fila.c src/filaPrior.c

# Run the exe
$ ./priority_list.exe
```
